# Сайн байна уу, би chikachimka8-cyber

Кибер аюулгүй байдал, програм хөгжүүлэлтэд сонирхолтой. Код бичиж сурч, төслүүдээ энд хуваалцаж байна.

---

## Миний тухай

- Кибер аюулгүй байдал, програмчлалд дуртай
- Жижиг төсөл, скрипт хийж сурч байна
- Энэ репозитори дээр туршилтын төслүүдээ байрлуулна
